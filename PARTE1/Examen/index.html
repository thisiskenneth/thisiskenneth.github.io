Boostrap

Bootstrap es un framework de desarrollo front-end open-source creado por el equipo de Twitter. Está diseñado para facilitar la creación de sitios web y aplicaciones web responsivas y móviles, utilizando una combinación de HTML, CSS y JavaScript. La versión más reciente, Bootstrap 5, elimina las dependencias de jQuery y ofrece un soporte mejorado para CSS personalizado.


Sistema de Grillas:
El sistema de grillas de Bootstrap es uno de sus componentes más potentes. Utiliza un diseño de rejilla de 12 columnas que se ajusta automáticamente a diferentes tamaños de pantalla. Puedes combinar clases como .col- y .col-md- para definir cómo se distribuirá el contenido en diferentes dispositivos.


Componentes:
Bootstrap incluye una amplia variedad de componentes predefinidos que se pueden integrar fácilmente en cualquier proyecto. Algunos de los componentes más utilizados incluyen:

  • Botones: Clases como .btn y .btn-primary permiten crear botones con estilos consistentes.
  • Formularios: Bootstrap proporciona estilos para formularios, incluyendo clases para controles de formulario, etiquetas y validación de formularios.
  • Tarjetas: Las tarjetas son contenedores flexibles y extensibles que pueden contener una variedad de contenido.
  • Modales: Los modales son ventanas emergentes que se pueden usar para diálogos, alertas o formularios.
  • Navbars: Las barras de navegación responsivas se pueden crear usando las clases .navbar y .navbar-expand-.
  • Alertas: Mensajes de alerta con clases como .alert y .alert-warning para mostrar información importante al usuario.


Utilidades:
Bootstrap también ofrece una amplia gama de utilidades para espaciado, alineación, colores, sombras y más. Estas utilidades permiten aplicar estilos específicos a elementos HTML sin necesidad de escribir CSS personalizado.

  • Espaciado: Clases como .m-3 y .p-3 se utilizan para añadir márgenes y relleno a los elementos.
  • Alineación: Clases como .text-center y .align-items-center ayudan a centrar el texto y alinear los elementos.
  • Colores: Clases para colores de fondo y texto como .bg-primary y .text-secondary.
  • Sombras: Clases como .shadow y .shadow-lg añaden sombras a los elementos para darles profundidad.


JavaScript:
Bootstrap incluye una biblioteca de JavaScript que proporciona componentes interactivos como modales, pestañas, tooltips, popovers, alertas y carousels. Estos componentes se pueden activar mediante atributos de datos (data attributes) o usando una API JavaScript.


Personalización:
Una de las características más poderosas de Bootstrap es su capacidad de personalización. Puedes descargar Bootstrap y modificar las variables SASS para cambiar los colores, tipografías, tamaños y otros aspectos del diseño. Bootstrap también incluye un generador de temas que permite personalizar el aspecto visual de los componentes.


Documentación y Comunidad:
Bootstrap tiene una documentación extensa y bien organizada que facilita a los desarrolladores aprender y utilizar el framework. Además, cuenta con una comunidad activa que contribuye con tutoriales, temas y soporte en foros y redes sociales.

Grid System

El sistema de grillas de Bootstrap es una de las características más potentes y flexibles del framework. Permite crear diseños responsivos utilizando un contenedor y una serie de filas y columnas.


El sistema de grillas de Bootstrap se basa en un diseño de 12 columnas. Esto significa que puedes dividir el contenedor en hasta 12 partes iguales, y combinar columnas para crear diseños de cualquier tamaño. Las clases de columna utilizan el prefijo .col- seguido por el número de columnas que debe ocupar un elemento.


Contenedores:
Los contenedores proporcionan un medio para alinear correctamente el contenido en el diseño. Bootstrap ofrece dos tipos de contenedores:

  • .container: Un contenedor fijo con un ancho máximo en cada punto de interrupción (breakpoint).
  • .container-fluid: Un contenedor de ancho completo, que abarca el 100% del ancho de la ventana del navegador.


Filas:
Las filas (.row) son necesarias para crear columnas. Se utilizan dentro de los contenedores para agrupar las columnas y garantizar el alineamiento adecuado.


Columnas:
Las columnas se crean dentro de las filas y se definen utilizando clases como .col-. Bootstrap incluye clases de columnas para cada punto de interrupción de diseño responsivo:

  • .col-: Para dispositivos extra pequeños (ancho menor a 576px).
  • .col-sm-: Para dispositivos pequeños (576px y mayores).
  • .col-md-: Para dispositivos medianos (768px y mayores).
  • .col-lg-: Para dispositivos grandes (992px y mayores).
  • .col-xl-: Para dispositivos extra grandes (1200px y mayores).

Botones

Los botones en Bootstrap son elementos versátiles que pueden ser utilizados en formularios, diálogos, y otros componentes interactivos. Bootstrap proporciona una amplia variedad de estilos de botones que se pueden aplicar mediante clases predefinidas.


Clases de Botones:
Bootstrap ofrece varias clases de botones para diferentes propósitos y niveles de importancia. Aquí están las clases básicas que puedes utilizar:

  • .btn: Clase base para todos los botones.
  • .btn-primary: Botón de acción principal, generalmente usado para acciones importantes.
  • .btn-secondary: Botón de acción secundaria.
  • .btn-success: Botón utilizado para indicar una acción exitosa.
  • .btn-danger: Botón utilizado para advertencias o acciones destructivas.
  • .btn-warning: Botón utilizado para advertencias.
  • .btn-info: Botón utilizado para información general.
  • .btn-light: Botón con estilo claro, adecuado para fondos oscuros.
  • .btn-dark: Botón con estilo oscuro, adecuado para fondos claros.
  • .btn-link: Botón estilizado como un enlace, ideal para acciones menos importantes.

Formularios con Bootstrap

Bootstrap simplifica la creación y el diseño de formularios en páginas web mediante el uso de clases predefinidas. Estas clases permiten mejorar la apariencia y la funcionalidad de los formularios, haciéndolos más atractivos y fáciles de usar.


Clases Básicas:
- .form-group: Esta clase se utiliza para agrupar etiquetas <label> y elementos de formulario, como <input> o <select>. Ayuda a organizar y alinear correctamente los elementos del formulario.
- .form-control: Esta clase se aplica a elementos de formulario como <input>, <select>, y <textarea>. Proporciona estilos de borde, relleno y tamaño uniformes para estos elementos, creando un aspecto coherente en todo el formulario.
- .btn: Esta clase se utiliza para estilizar los botones en el formulario. Proporciona un aspecto atractivo y uniforme a los botones, haciéndolos más fáciles de identificar y utilizar.

NavBar

Bootstrap ofrece una barra de navegación totalmente personalizable y responsiva, conocida como NavBar. Esta barra de navegación puede contener enlaces, formularios, texto, y otros elementos de navegación, y se adapta automáticamente a diferentes tamaños de pantalla.


Estructura Básica:
La estructura básica de una NavBar comienza con un contenedor <nav> con la clase .navbar. Dentro de este contenedor, puedes añadir varios elementos como enlaces y botones.



Modal

Los modales en Bootstrap son ventanas emergentes que se superponen en la página principal para mostrar contenido adicional, como mensajes, formularios o imágenes. Estas ventanas emergentes son útiles para interactuar con el usuario sin necesidad de redireccionar a otra página.


Componentes Principales:
- .modal: Esta es la clase base que se utiliza para crear un modal en Bootstrap. Define las propiedades básicas del modal, como el tamaño, la posición y el efecto de animación.
- .modal-dialog: Esta clase define el contenedor del modal y establece su anchura. Se utiliza para controlar el tamaño del modal, como pequeño, mediano o grande.
- .modal-content: Esta clase se utiliza para definir el contenido del modal, incluyendo el encabezado, el cuerpo y el pie. Ayuda a estructurar y organizar el contenido dentro del modal.
- .modal-header: Esta clase se utiliza para agregar un encabezado al modal. Puede contener un título y un botón para cerrar el modal.
- .modal-body: Esta clase se utiliza para agregar el contenido principal al modal. Puede contener texto, formularios, imágenes u otros elementos HTML.
- .modal-footer: Esta clase se utiliza para agregar un pie de página al modal. Puede contener botones u otros elementos de acción para interactuar con el contenido del modal.

Glyphicon proporcionados por Bootstrap 3

-Visualizamos los glyphicon con las clases "glyphicon glyphicon-*" donde " * " es el tipo de glyphicon; nosotros usamos "user", "book", "signal", "hdd" y "fire". Para nuestro ejemplo visualizaremos los glyphicon utilizando la etiqueta "span" y agregando sus respectivas clases.




Tamaño y color a los Glyphicon

-Para cambiar el tamaño y color a los glyphicon es necesario aplicar estilos, para eso vamos a la hoja de estilos main.css y creamos las clases "app", "icon-color1", "icon-color2", "icon-color3", "icon-color4" y "icon-color5". A la clase "app" le agregamos un "font-size:50px" para darle un tamaño de 50 pixeles "padding-bottom: 10px;" para separarlos un poco de los demas elementos; a las demás clases le agregamos "color:#xxxxxx" y le definimos a cada uno un color.

Al ser los Glyphicon's considerados texto, podemos realizar cambios en sus estilos como cambiar tamaño, cambiar color, colocarle un color de fondo, rotarlo, colocarle sombras y muchas cosas más en CSS3.




Descargar e insertar glyphicon externos - PARTE II

Podemos descargar e insertar glyphicon que nosotros obtegamos de deferentes fuentes, Nosotros le proporcionamos un ejemplo y todos los pasos a seguir:

  • Visitar la página http://icomoon.io/app/#/select.
  • Damos clic en el botón "font" y finalmente en el bootón "Dowload".
  • Descomprimimos el archivo.
  • Copiamos los archivos dentro de la carpeta "fond" de nuestros archivos ddescargados y los llevamos a la carpeta "fond" de nuestro proyecto.
  • Copiamos el archivo "style" y lo copiamos dentro de la carpeta "css" de nuestro proyecto y lo cambiamos de nombre por "iconbar".
  • En nuestro sublimetext abrimos el archivo icon bar.
  • Modificaremos la ruta de los Glyphicon en cada URL ateponiendo " ..// "
  • Y para finalizar, en el archivo HTML en lazaremos nuestra hoja de estilo "css/iconbar.css"

    • Evitar declarar clases con que comiencen con "icon-" en nuestro "main.css" ya que habrá un conflicto con la hoja de estilo "iconbar.css".





    Glyphicon en Botones y formularios - PARTE III
